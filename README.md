# admin-projects-npa

### Информация

Модуль проектов НПА заточенный для CMS IRsite.

### Установка

```
$ composer require avl/admin-projects-npa
```
Или в секцию **require** добавить строчку **"avl/admin-projects-npa": "^1.0"**

```json
{
    "require": {
        "avl/admin-projects-npa": "^1.0"
    }
}
```
